```python
for i in range(10):
    #loops from 0 to 9
```

```python
for i in range(5, 10): 
    # loop from 5 to 9
```

```python
for i in range(4, 20, 2): 
    # loop from 4 to 18, skipping by 2 each time
```

```python
for i in range(10, 0, -1): 
    # Step backward from 10 to 1
```

### with Strings
```python
s="hello"

for c in s
    print(c)
```

### with Lists
```python
x=[1,2,3,4,5]

for i in x
    print(i)
```

```python
x=[1,2,3,4,5]

for i in range(len(x))
    print(x[i])
```
